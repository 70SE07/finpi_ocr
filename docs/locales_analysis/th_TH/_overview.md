# Таиланд (th_TH)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | th_TH |
| Язык | Тайский (Thai) |
| Письменность | Thai script |
| Валюта | THB (Thai Baht) |
| Символ валюты | ฿ или "บาท" (baht) |
| Десятичный разделитель | . (точка) |
| Разделитель тысяч | , (запятая) |
| Формат суммы | 1,234.56 |

## Форматы даты

- **Тайский буддийский календарь:** год = григорианский + 543
- `DD/MM/YY` - 03/10/68 (= 03/10/2025)
- Время: `HH:MM` - 23:33

## Налоговая система (VAT)

| Параметр | Значение |
|----------|----------|
| Ставка НДС | 7% |
| Отображение | "VAT Included" - включен в цену |
| Tax block | Отсутствует |

## Типичная структура чека (7-Eleven)

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - CP ALL, 7-Eleven [название филиала]   │
│ - TAX# (налоговый номер)                │
│ - Vat Code, POS#                        │
├─────────────────────────────────────────┤
│ ITEMS                                   │
│ Qty  Название             @price  Total │
│  3   ผัดมักกะโรนีไก่        @37.00  111.00│
├─────────────────────────────────────────┤
│ SUBTOTAL                                │
│ ถลรวม (subtotal)                 783.25 │
├─────────────────────────────────────────┤
│ DISCOUNTS (отдельный блок)              │
│ ส่วนลด[название]                  20.00 │
├─────────────────────────────────────────┤
│ NET TOTAL                               │
│ ถลสุทธิ [qty] ชิ้น              758.25  │
├─────────────────────────────────────────┤
│ PAYMENT                                 │
│ เงินสด/เงินทอน  1,000.00    241.75      │
├─────────────────────────────────────────┤
│ FOOTER                                  │
│ R#[номер] [дата] [время]                │
│ ** ศูนย์ลูกค้าสัมพันธ์ [телефон] **      │
│ บิลนี้ประหยัด [сумма экономии]           │
│ [QR код]                                │
└─────────────────────────────────────────┘
```

## Ключевые слова

### Итоговая сумма
- ถลรวม (subtotal)
- ถลสุทธิ (net total)
- รวม (total)

### Скидки
- ส่วนลด (discount)

### Оплата
- เงินสด (cash)
- เงินทอน (change)

### Количество
- ชิ้น (pieces/items)
- ตอละ / ตัวละ (each/per piece)

## Особенности для парсинга

1. **Thai script** - требуется поддержка Unicode
2. **Qty в начале строки** - отличается от европейских чеков
3. **"@" как маркер unit_price** - вместо "x"
4. **Нет tax block** - только общая сумма с VAT included
5. **Буддийский календарь** - год +543 от григорианского
6. **Скидки в отдельном блоке** - не рядом с товаром
7. **Бесплатные марки (stamps)** - программа лояльности, цена 0.00
