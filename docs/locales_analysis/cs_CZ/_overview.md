# Чехия (cs_CZ)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | cs_CZ |
| Язык | Чешский (Czech) |
| Валюта | CZK (Koruna česká) |
| Символ валюты | Kč или CZK |
| Десятичный разделитель | , (запятая) |
| Разделитель тысяч | пробел |
| Формат суммы | 1 135,78 |

## Форматы даты

- `DD.MM.YYYY` - 26.12.2025
- Время: `HH:MM:SS` - 18:05:26

## Налоговая система (DPH)

| Ставка | Код | Применение |
|--------|-----|------------|
| 21% | 1 | Стандартная ставка |
| 15% | 2 | Пониженная ставка |
| 10% | 3 | Супер-пониженная ставка |

## Особенность: Zaokrouhlení (округление)

В Чехии суммы округляются до целых крон:
- SOUČET: 1135,78 CZK
- Zaokrouhlení: +0,22 CZK
- K zaplacení: 1136,00 CZK

**Это важно для checksum валидации!**

## Типичная структура чека

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - Название магазина                     │
│ - Адрес, Sídlo                          │
│ - DIČ (налоговый номер)                 │
├─────────────────────────────────────────┤
│ ITEMS                                   │
│ Art/Ean [код]                           │
│ [qty] [unit] x [price]                  │
│ [название]                     [total] 1│
├─────────────────────────────────────────┤
│ SOUČET [21]    CZK        1135,78       │
│ Zaokrouhlení               0,22         │
│ K zaplacení:              1136,00       │
├─────────────────────────────────────────┤
│ TAX BLOCK                               │
│      BRUTTO    DPH     NETTO            │
│ 1 21%  1135,78  197,12  938,66          │
├─────────────────────────────────────────┤
│ FOOTER                                  │
│ - Čas evidence tržby                    │
│ - BKP, FIK (фискальные коды)            │
└─────────────────────────────────────────┘
```

## Ключевые слова

### Итоговая сумма
- součet, celkem, k zaplacení, k úhradě

### Скидки
- sleva, akce

### Округление
- zaokrouhlení

### Оплата
- hotovost (наличные), karta (карта), vráceno (сдача)

## Особенности для парсинга

1. **Zaokrouhlení** - округление до целых крон (влияет на checksum)
2. **Art/Ean** - артикул на отдельной строке перед товаром
3. **Многострочный формат** - qty/unit/price на отдельной строке
4. **Tax code "1"** в конце строки (не буква как в Германии)
5. **Весовые товары** - `0,020 kg x 613,00`

## Проблемные случаи

### Чек без шапки (cz_001.jpeg)
- Клиент сфотографировал без верхней части
- Нельзя определить магазин
- **Решение:** Fallback на определение по косвенным признакам (формат, ключевые слова)
