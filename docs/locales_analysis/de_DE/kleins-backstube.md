```

excl. 7% Mwst           4,39 €
7% Mwst                 0,31 €
```

**Проверка:**
- excl. 7% = Netto = 4,39
- 7% MwSt = 0,31
- 4,39 × 0.07 = 0,31 ✓
- 4,39 + 0,31 = 4,70 ✓

**Важно:** 7% MwSt на выпечку (продукты питания)
