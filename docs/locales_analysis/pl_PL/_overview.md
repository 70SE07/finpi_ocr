# Польша (pl_PL)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | pl_PL |
| Язык | Польский (Polish) |
| Валюта | PLN (złoty) |
| Символ валюты | zł или PLN |
| Десятичный разделитель | , (запятая) ИЛИ . (точка)! |
| Разделитель тысяч | пробел |
| Формат суммы | 254,32 или 50.97 |

## ВАЖНО: Два формата чисел!

В Польше встречаются ОБА формата:
- **Запятая:** 254,32 (Carrefour, Cropp)
- **Точка:** 50.97 (Orlen)

## Форматы даты

- `YYYY-MM-DD` - 2025-11-19
- `DD-MM-YYYY` - 21-11-2025
- Время: `HH:MM` - 18:10

## Налоговая система (PTU/VAT)

| Класс | Ставка | Применение |
|-------|--------|------------|
| A | 23% | Стандартная ставка |
| B | 8% | Пониженная (продукты) |
| C | 5% | Супер-пониженная |
| D | 0% | Освобожденные |

## Типичная структура чека

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - Название магазина                     │
│ - Адрес                                 │
│ - NIP (налоговый номер)                 │
│ - nr wydr. (номер чека)                 │
├─────────────────────────────────────────┤
│ [дата]         PARAGON FISKALNY         │
├─────────────────────────────────────────┤
│ ITEMS                                   │
│ [prefix]_[название]  [qty]*[price]=[total] [tax]│
│ Пример:                                 │
│ C_POMELO CZERWONE L  0,762*9,99= 7,61 C │
├─────────────────────────────────────────┤
│ TAX BLOCK                               │
│ Sprzed. opod. PTU A:         99,55      │
│ Kwota A 23,00%:              18,62      │
│ Sprzed. opod. PTU C:        154,77      │
│ Kwota C 05,00%:               7,37      │
│ Podatek PTU:                 25,99      │
├─────────────────────────────────────────┤
│ SUMA PLN                    254,32      │
├─────────────────────────────────────────┤
│ FOOTER                                  │
│ - Штрих-код                             │
│ - Платежные данные                      │
│ - PAYBACK                               │
└─────────────────────────────────────────┘
```

## Ключевые слова

### Итоговая сумма
- suma, razem, do zapłaty, kwota

### Налог
- PTU, VAT, sprzed. opod., kwota, podatek

### Оплата
- karta kredytowa, gotówka, płatność

### Отмена
- anulowano sprzedaż (отмена продажи)

## Особенности для парсинга

1. **Префикс категории** - `A_`, `C_` перед названием
2. **Формат qty*price=total** - компактный
3. **Tax class в конце** - A, B, C, D
4. **Tax block** - детализация PTU
5. **PARAGON FISKALNY** - фискальный чек (обязательная надпись)
6. **Два формата чисел** - , или . как десятичный разделитель
7. **Отмена товара** - `***Anulowano sprzedaż***`
