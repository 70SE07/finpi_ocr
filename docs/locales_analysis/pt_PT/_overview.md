# Португалия (pt_PT)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | pt_PT |
| Язык | Португальский (Portuguese) |
| Валюта | EUR |
| Символ валюты | € (после суммы) |
| Десятичный разделитель | , (запятая) |
| Разделитель тысяч | . (точка) или пробел |
| Формат суммы | 1.234,56 или 1 234,56 |

## Форматы даты

- `DD/MM/YYYY` - 09/11/2025
- `DD-MM-YYYY` - 09-11-2025
- Время: `HH:MM` - 13:10

## Налоговая система (IVA)

| Класс | Ставка | Применение |
|-------|--------|------------|
| (A) | 6% | Базовые продукты (Bens Essenciais) |
| (B) | 13% | Промежуточная ставка |
| (C) | 23% | Стандартная ставка |

## Ключевые слова

### Итоговая сумма
- total, total a pagar, valor

### Скидки
- poupanca, desconto, promoção

### Категории товаров
- Laticinios/Beb. Veg. (молочные/растительные напитки)
- Bens Essenciais (базовые товары)
- Charcutaria&Queijos (колбасы и сыры)
- Frutas e Legumes (фрукты и овощи)

### Оплата
- cartao credito, multibanco, dinheiro

## Особенности для парсинга

1. **Tax class (A/B/C)** перед названием товара
2. **Категории товаров** как разделители в чеке
3. **Весовой товар** - детализация на отдельной строке (2,200 X 3,99)
4. **POUPANCA** - скидка (может быть уже учтена в цене)
5. **Tax block есть** - можно использовать для валидации
6. **Fatura Simplificada** - упрощенный счет (до определенной суммы)
7. **MULTIBANCO** - национальная платежная система
8. **ATCUD** - обязательный налоговый код
