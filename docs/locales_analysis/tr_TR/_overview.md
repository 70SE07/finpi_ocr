# Турция (tr_TR)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | tr_TR |
| Язык | Турецкий (Turkish) |
| Валюта | TRY/TL (Turkish Lira) |
| Символ валюты | TL или ₺ |
| Десятичный разделитель | , (запятая) |
| Разделитель тысяч | . (точка) |
| Формат суммы | 1.234,56 |

## Форматы даты

- `DD/MM/YYYY` - 13/10/2025
- Время: `HH:MM:SS` - 13:43:08

## Налоговая система (KDV)

| Ставка | Применение |
|--------|------------|
| 20% | Стандартная ставка |
| 10% | Продукты питания, рестораны |
| 1% | Базовые продукты |

## Типичная структура чека

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - Название (F33 ENVA)                   │
│ - Адрес                                 │
│ - V.D (налоговый номер)                 │
├─────────────────────────────────────────┤
│ TARİH   : DD/MM/YYYY                    │
│ SAAT    : HH:MM:SS                      │
│ FİŞ NO  : [номер]                       │
├─────────────────────────────────────────┤
│ ITEMS                                   │
│ [название]    %[tax]    *[цена]         │
├─────────────────────────────────────────┤
│ TOPKDV         *[сумма KDV]             │
│ TOPLAM         *[итого]                 │
├─────────────────────────────────────────┤
│ KREDİ          *[оплата картой]         │
│ [банк]                                  │
├─────────────────────────────────────────┤
│ FOOTER                                  │
│ - SATIŞ, TUTAR                          │
│ - Банковский терминал                   │
└─────────────────────────────────────────┘
```

## Ключевые слова

### Итоговая сумма
- toplam, tutar

### Налог
- topkdv, kdv

### Оплата
- kredi (карта), nakit (наличные)

### Дата/время
- tarih (дата), saat (время)

## Особенности для парсинга

1. **Звездочка перед ценой** - `*85,00`
2. **Процент налога в строке** - `%10`
3. **TOPKDV** - общая сумма НДС
4. **TOPLAM** - итого
5. **Турецкие буквы** - İ, Ş, Ü, Ö, Ç, Ğ
