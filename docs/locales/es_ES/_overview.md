# Испания (es_ES)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | es_ES |
| Язык | Испанский (Spanish) |
| Валюта | EUR |
| Символ валюты | € (после суммы) |
| Десятичный разделитель | , (запятая) |
| Разделитель тысяч | . (точка) |
| Формат суммы | 1.234,56 |

## Форматы даты

- `DD.MM.YYYY` - 28.10.2025
- `DD/MM/YYYY` - 28/10/2025
- Время: `HH:MM` - 18:40

## Налоговая система (IVA)

| Ставка | Применение |
|--------|------------|
| 21% | Общая ставка |
| 10% | Продукты питания, транспорт |
| 4% | Базовые продукты |

**Примечание:** В чеках Consum налоговый блок может отсутствовать.

## Ключевые слова

### Итоговая сумма
- total, total factura, importe, importe a abonar, a pagar

### Скидки
- descuento, dto, oferta, promoción

### Оплата
- tarjeta, tarj. crédito, efectivo, cambio

### Количество
- und (unidades), kg, gr, l, ml

## Особенности для парсинга

1. **Qty в начале строки** - как в Таиланде
2. **Весовые товары** - qty с запятой (0,580)
3. **Заголовки колонок** - UND, ARTICULO, PVP, €/KG, TOTAL
4. **Tax block** - может отсутствовать
5. **Loyalty программа** - Tarjeta MundoConsum
