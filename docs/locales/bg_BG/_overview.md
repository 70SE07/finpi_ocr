# Болгария (bg_BG)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | bg_BG |
| Язык | Болгарский (Bulgarian) |
| Письменность | Кириллица |
| Валюта | BGN (лев) |
| Символ валюты | лв. |
| Десятичный разделитель | . (точка) |
| Разделитель тысяч | нет или пробел |
| Формат суммы | 26.13 |

## Форматы даты

- `DD.MM.YYYY` - 01.09.2025
- Время: `HH:MM:SS` - 17:29:57

## Налоговая система (ДДС)

| Ставка | Код | Применение |
|--------|-----|------------|
| 20% | Б | Стандартная ставка |
| 9% | А | Пониженная (туризм) |

## Курс к евро

На чеках часто показывается эквивалент в евро:
```
ОБЩА СУМА        ЛВ.    26.13
ОБЩА СУМА        ЕВРО   13.36
Курс 1 ЕВРО = 1.95583 ЛВ.
```

## Типичная структура чека

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - БИЛЛА БЪЛГАРИЯ ЕООД                   │
│ - Адрес (СОФИЯ бул. БЪЛГАРИЯ)           │
│ - ЕИК (идентификационный номер)         │
│ - НОМЕР ЗДДС                            │
├─────────────────────────────────────────┤
│ ITEMS                                   │
│ [название]            *Б     [цена]     │
│ ОТСТ. СУПЕР ЦЕНА      *Б    -[скидка]   │
├─────────────────────────────────────────┤
│ ОБЩА СУМА          ЛВ.      26.13       │
│ ОБЩА СУМА          ЕВРО     13.36       │
│ Курс 1 ЕВРО = 1.95583 ЛВ.               │
├─────────────────────────────────────────┤
│ ПЛАТЕНО В БРОЙ     ЛВ.      30.00       │
│ РЕСТО              ЛВ.       3.87       │
│ #ТИ СПЕСТИ:        ЛВ.       2.30#      │
├─────────────────────────────────────────┤
│ FOOTER                                  │
│ [дата] [время] [артикулов]              │
│ СИСТЕМЕН БОН                            │
│ [QR код]                                │
└─────────────────────────────────────────┘
```

## Ключевые слова

### Итоговая сумма
- обща сума, общо, сума, за плащане

### Скидки
- отстъпка, отст., акция, супер цена

### Оплата
- платено в брой (наличные), ресто (сдача)

### Экономия
- ти спести (ты сэкономил)

## Особенности для парсинга

1. **Кириллица** - болгарский алфавит
2. **Tax class "*Б"** в конце строки (со звездочкой)
3. **Скидки** идут сразу после товара
4. **Две валюты** - показаны BGN и EUR эквивалент
5. **#ТИ СПЕСТИ#** - общая экономия в специальном блоке
