# Индия (en_IN)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | en_IN (английский) |
| Язык | Английский / Хинди |
| Валюта | INR (Indian Rupee) |
| Символ валюты | ₹ или Rs. |
| Десятичный разделитель | . (точка) |
| Разделитель тысяч | , (запятая) |
| Формат суммы | 1,234.56 |

## Форматы даты

- `DD-MM-YYYY` - 30-09-2025
- Время: `HH:MM` - 07:08

## Налоговая система (GST)

Индия использует **GST (Goods and Services Tax)**:

| Компонент | Ставка | Описание |
|-----------|--------|----------|
| CGST | 2.5% | Central GST |
| SGST | 2.5% | State GST |
| **Total GST** | **5%** | Итого |

Или:
| Компонент | Ставка |
|-----------|--------|
| CGST | 9% |
| SGST | 9% |
| **Total GST** | **18%** |

```
┌─────────────────────────────────────────┐
│ ** TAX INVOICE **                       │
├─────────────────────────────────────────┤
│ HEADER                                  │
│ - Название                              │
│ - Адрес                                 │
│ - PIN (почтовый индекс)                 │
│ - GSTIN (налоговый номер)               │
├─────────────────────────────────────────┤
│ Receipt No.: [номер]                    │
│ Employee...: [имя]                      │
│ Date.: [дата] Time.: [время]            │
├─────────────────────────────────────────┤
│ ITEM NAME   QTY  PRICE   Total          │
│ ----------------------------------------│
│ [название]                              │
│ [код]  [qty]  [price]   [total]         │
├─────────────────────────────────────────┤
│ Tax Summary                             │
│ Taxable value  Tax %  Tax Amount        │
│    428.57      2.50     10.71           │
│    428.57      2.50     10.71           │
│ Subtotal                    428.58      │
│ Total tax                    21.42      │
│ Total                       450.00      │
├─────────────────────────────────────────┤
│ Card                        450.00      │
│ Thanks for shopping!                    │
└─────────────────────────────────────────┘
```

## Особенности для парсинга

1. **Английский язык** - легко парсить
2. **GSTIN** - налоговый идентификатор (формат: 27AAPFN0717N1ZT)
3. **Tax Summary** - детализация CGST + SGST
4. **Табличный формат** - ITEM NAME QTY PRICE Total
5. **Subtotal + Tax = Total**
