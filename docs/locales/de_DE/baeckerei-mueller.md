```

  %          €          €         €

```

**Проверка:**
- Netto: 4,92
- MwSt: 0,93
- 4,92 × 0.19 = 0,93 ✓
- 4,92 + 0,93 = 5,85 ✓

## Особенности

1. **Tax class "C"** - не A/B как в супермаркетах!
2. **Rechnung формат** - табличный с заголовками
3. **Многострочный** - название и цена на разных строках
4. **Stück** - единица измерения всегда указана
5. **Пекарня/кафе** - другой формат чем ритейл

## КРИТИЧЕСКОЕ НАБЛЮДЕНИЕ

**В Германии tax class может быть C!**

| Класс | Ставка | Где встречается |
|-------|--------|-----------------|
| A | 7% | Супермаркеты (продукты) |
| B | 19% | Супермаркеты (непрод) |
| C | 19% | Пекарни, кафе |
| 1 | 19% | dm (числовой код) |
