# Украина (uk_UA)

## Базовая информация

| Параметр | Значение |
|----------|----------|
| Код локали | uk_UA |
| Язык | Украинский (Ukrainian) |
| Письменность | Кириллица |
| Валюта | UAH (гривня) |
| Символ валюты | ГРН или ₴ |
| Десятичный разделитель | , (запятая) |
| Разделитель тысяч | пробел |
| Формат суммы | 15 305 або 15 305,00 |

## Форматы даты

- Григорианский календарь
- `DD.MM.YYYY` - 26.12.2025
- Время: `HH:MM` - 14:30

## Налоговая система (ПДВ)

| Класс | Ставка | Применение |
|-------|--------|------------|
| Я | 20% | Стандартная ставка ПДВ |
| - | 0% | Без ПДВ (неоп) |

**Примечание:** На чеках может быть "Без ПДВ Я = неоп" - товары без НДС.

## Типичная структура чека

```
┌─────────────────────────────────────────┐
│ HEADER                                  │
│ - Логотип магазина                      │

│ - Каса N                                │
├─────────────────────────────────────────┤
│ ITEMS (многострочный формат)            │
│ 1 шт X 234 =                            │

│ Знижка                                  │

│ BONUSES                                 │

```

## Ключевые слова

### Итоговая сумма
- сума, всього, разом, до сплати

### Скидки
- знижка, знижки, акція

### Оплата
- готівка, безготівкова, картка

### Количество
- шт (штук), кг, г, л, мл

## Особенности для парсинга

1. **Многострочный формат товара:**
   - Строка 1: `1 шт X [цена] =`
   - Строка 2-N: Название (может быть несколько строк)
   - Последняя: `[итого] Я`
2. **Знижка** (скидка) сразу после товара
3. **Tax class "Я"** в конце строки с ценой
4. **Целые числа** - без копеек для крупных сумм
5. **Пробел** как разделитель тысяч (15 305)
6. **Бонусная система** - карта лояльности
7. **Кириллица** - украинский алфавит
